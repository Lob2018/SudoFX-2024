Detailed Description

Utility class for managing JVM application properties.

This class provides methods to retrieve the application name and version from system properties.

◆ isSpringContextExitOnRefresh()

static boolean fr.softsf.sudofx2024.utils.JVMApplicationProperties.isSpringContextExitOnRefresh ( )
static

Checks if the Spring context should exit on refresh.

If the 'springContextExit' variable is empty, it retrieves the value from the system property specified by SPRING_CONTEXT_EXIT_ON_REFRESH. The method returns true if the value is "onRefresh"; otherwise, it returns false.

Returns
true if the context should exit on refresh; false otherwise.
