Flyway fr.softsf.sudofx2024.utils.database.configuration.DynamicDataSourceConfiguration.flyway ( HikariDataSource dataSource)
inline

Configures Flyway with a specified location for migration scripts.

This method initializes a Flyway instance that will manage database migrations using the provided data source. It specifies the location of the migration scripts which Flyway will execute to ensure the database schema is up-to-date.

Parameters
dataSourcethe HikariDataSource used by Flyway to connect to the database. This data source should be properly configured with the necessary connection details (URL, username, password).
Returns
a Flyway instance configured with the specified data source and migration script location. The initMethod "migrate" will be called automatically after the bean is created, applying any pending migrations to the database.
